Soccer Odds (content zipped using Gzip compression)

http://www.goalserve.com/getfeed/59e68707a50347e8fa7508de75e150c9/getodds/soccer?cat=soccer_10

a)Timestamp - Import only latest changes after last download, instead of loading full feed.

<scores sport="soccer" ts="1474825423341">

http://www.goalserve.com/getfeed/.../getodds/soccer?cat=soccer_10&ts=1474825423341

b)Feed compressed using GZIP 
c)Filter parameters
     date_start - start date filter
     date_end - end date filter. Can be null if you need to get matches just for 1 day - use only date_start in this case
     bm - filter by bookmaker (comma separated)
     market- filter by market (comma separated)
     league - filter by category id (comma separated). If you want to filter by "gid" attribute add   "_"        symbol before gid value. For exame "&league=_2343,_2435".
     match - filter by match id (comma separated)
